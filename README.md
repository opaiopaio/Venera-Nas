# Venera - NAS
> **本项目基于 [haukuen/Venera](https://github.com/haukuen/Venera) 二次开发，在此向原作者致以诚挚的感谢！**

## 📌 与原版的区别

| SMB 协议连接 NAS 
| 从 NAS 扫描漫画
| 下载到 NAS (SMB)
| SMB 漫画流式阅读
| 两种下载方式（本地 / NAS）

## 🚀 快速开始

### 配置 SMB 服务器
1. 进入 **设置 → 网络 → SMB / NAS Servers**
2. 点击 **Add Server**，填写 NAS 信息（主机、共享名、用户名、密码）
3. 点击 **Test Connection** 验证连接

### 从 NAS 导入漫画
- 在主页点击 导入旁的扫描按钮，或者进入本地页面点击右上角
- 选择服务器和根目录，开始扫描

### 下载方式选择
- 下载章节时，可选择 **下载到本地** 或 **下载到 NAS (SMB)**
- 可在设置中配置默认下载方式
